# Static Images Directory

This folder is created for static image assets. Since it sits inside the \`public\` directory, any files here will be served at the root path during runtime.

For example, if you place \`logo.png\` here (e.g., \`public/assets/images/logo.png\`), it can be accessed in your app using:
\`<img src="/assets/images/logo.png" />\`

You can upload your files here directly via the code editor's file explorer.
